data.met.no

The data.met.no API provides free access to the MET Norway's archive of historical weather and climate data, in addition to station history information. These data include quality controlled daily, monthly, and yearly measurements of temperature, precipitation, and wind data.

NOTE: The data.met.no API is currently in the early stages of development, so expect rapid changes to any and all services made available on these pages.

Getting Started

This API is primarily for developers who need to develop scripts or applications that use MET Norway's archive of historical weather and climate data. An API key is required to use the API. There are currently no hard data retrieval limits on API keys, but any API keys which place undue strain on the server infrastructure during this period will be deleted without prior warning.

To get started using the data.met.no API:

  1. Request an API-Key.
  2. Play with and learn more about the resources available in Swagger (use the API-Key).
  3. Begin developing your application. All calls must be authenticated with your Api-Key.
