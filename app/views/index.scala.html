data.met.no

The data.met.no API provides free access to MET Norway's archive of historical weather and climate data, in addition to station history information. These data include quality controlled daily, monthly, and yearly measurements of temperature, precipitation, and wind data.

NOTE: The data.met.no API is currently in the early stages of development, so expect rapid and significant changes to any and all resources made available on this service.

Getting Started

This API is primarily for developers who need to develop scripts or applications that use MET Norway's archive of historical weather and climate data. An API-Key is required to use the API. There are currently no hard data retrieval limits on API-Keys, but any API-Keys which place undue strain on the server infrastructure during this period will be deleted without prior warning.

To get started using the data.met.no API:

  1. Request an API-Key.
  2. Play with and learn more about the resources available in Swagger (use the API-Key).
  3. Begin developing your application. All calls must be authenticated with your API-Key. To authorize access to protected resources and data, you also need an access token using the OAuth2 client credentials flow (as well as permission to access those resources). Note that the vast majority of data on this service is open, so the authorization flow should rarely be required.
