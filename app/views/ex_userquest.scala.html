User question

SOURCES (stations)

As a user I want information about stations:

  • I have a list of station numbers, how can I get information about these stations?
    https://data.met.no/sources/v0.jsonld?ids=SN18700,SN50500,SN100&types=SensorSystem
  • I want information about which stations that have been operating in a certain period
    https://data.met.no/sources/v0.jsonld?types=SensorSystem&validtime=1900-01-01/1905-01-01 
    • Use the TimeSpecification: https://data.met.no/concepts/index.html#time_specification 
  • I want information about stations within a polygon
    https://data.met.no/sources/v0.jsonld?types=SensorSystem&geometry=POLYGON ((10 60,10 65, 11 65, 10 60)) 
    • Use the geometry specification: https://data.met.no/concepts/index.html#geometry_specification 
  • I want stations for a certain country
    https://data.met.no/sources/v0.jsonld?types=SensorSystem&country=SE 
  • I do not know exactly the name of the station, but I want to find out if there are stations which contains some of this name
    https://data.met.no/sources/v0.jsonld?types=SensorSystem&name=Sval* 
  • I do not need all attributes in the output, how can I get less output? Specify the fields you need in the output in the "fields" parameter:
    https://data.met.no/sources/v0.jsonld?types=SensorSystem&country=SE&fields=name,geometry 
    "Supported fields: name, countyid, countyname, country, wmoidentifier, municipalityname, validto, validfrom, municipalityid, stationholder, countrycode, geometry, level"
  • I want to know about the stations owned by a certain stationholder:
    https://data.met.no/sources/v0.jsonld?types=SensorSystem&stationholder=STATENS VEGVESEN 

ELEMENTS

As a user I want information about weather elements (parameters, variables):

  • I want to retrieve all information about some elements: https://data.met.no/elements/v0.jsonld?ids=air_temperature,surface_air_pressure&lang=en-US 
  • I only know the name of the old parameter that I used to use before. What is the new name?
    https://data.met.no/elements/v0.jsonld?oldElementCodes=TAM,TAN,TAX,UU,UUM&lang=en-US 
  • I already have data from another source using the CF-convention(http://cfconventions.org/Data/cf-standard-names/43/build/cf-standard-name-table.html )  standard name and wonder if this is available from data.met.no:
    https://data.met.no/elements/v0.jsonld?cfStandardNames=air_pressure_at_sea_level&lang=en-US 
  • I want the information about the element in Norwegian (nynorsk)
    https://data.met.no/elements/v0.jsonld?cfStandardNames=air_pressure_at_sea_level&lang=nn-NO 
    Bokmål: https://data.met.no/elements/v0.jsonld?cfStandardNames=air_pressure_at_sea_level&lang=nb-NO 
  • I do not need all the information in the output, only id, name and unit:
    https://data.met.no/elements/v0.jsonld?ids=surface_air_pressure,air_temperature,mean(air_temperature P1M)&fields=id,name,unit&lang=en-US 
    Supported fields: name, cfconvention, description, legacymetnoconvention, id, codetable, unit

FREQUENCIES

Rainfall: Intensity - duration - frequency (IDF)

  • Which stations have calculated frequencies of rainfall intensity?
    https://data.met.no/frequencies/rainfall/availableSources/v0.jsonld 
  • And what are the data for one station?
    https://data.met.no/frequencies/rainfall/v0.jsonld?sources=SN18701 gives all combinations of returnperiod(frequency), duration (in minutes) and values (intensity in ls/Ha)
  • I want only 100 years return period for 10 minutes interval in mm:
    https://data.met.no/frequencies/rainfall/v0.jsonld?sources=SN18701&durations=10&frequencies=100&unit=mm 
  • I want IDF values for a place where there is no station, is it possible? If you use the source called idf_bma1km_v1 you can give in a location
    https://data.met.no/frequencies/rainfall/v0.jsonld?sources=idf_bma1km&location=POINT(9.5 59.1)&durations=10,30,60&frequencies=100&unit=mm 
  • See information about this source:
    https://data.met.no/sources/v0.jsonld?ids=idf_bma1km&types=InterpolatedDataset&validtime=2000-01-01 

CLIMATE NORMALS

Climate normals are values for a period of 30 years, decided by WMO. Available periods are 1931-1960 and 1961-1990 for some elements and sources on monthly and daily basis.

  • What kind of normal values can I get? https://data.met.no/climatenormals/available/v0.jsonld?
  • Which stations have normal values for mean temperature? https://data.met.no/climatenormals/available/v0.jsonld?elements=mean(air_temperature P1M) 
  • Which elements and normal periods are available for a certain station? https://data.met.no/climatenormals/available/v0.jsonld?sources=SN18500 
  • Which stations do have normal values for the 1931/1960 normal period? https://data.met.no/climatenormals/available/v0.jsonld?periods=1931/1960 
  • How can I get the monthly normal values for some stations e.g. mean temperature? https://data.met.no/climatenormals/v0.jsonld?sources=SN100,SN180&elements=mean(air_temperature P1M) 

RECORDS

This endpoint gives the "day record" for some elements (min and max temperature) for every month and every county.

  • How can I get the records for all counties in Norway? https://data.met.no/records/countyExtremes/v0.jsonld 
  • What is the maximum temperature for Vestfold county?
    https://data.met.no/records/countyExtremes/v0.jsonld?counties=vestfold&elements=*max* 
  • What has been the maximum temperature in Vestfold in January, July,August.
    https://data.met.no/records/countyExtremes/v0.jsonld?counties=vestfold&elements=*max*&months=1,7,8
  • In which municipalities exist records in Vestfold (independent of element):
    https://data.met.no/records/countyExtremes/v0.jsonld?counties=vestfold&fields=municipality 
  • ... for minimum values:
    https://data.met.no/records/countyExtremes/v0.jsonld?counties=vestfold&elements=*min*&fields=municipality 
  • For which elements do records exist? https://data.met.no/records/countyExtremes/v0.jsonld?fields=elementid
